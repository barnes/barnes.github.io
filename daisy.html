CS50Web Project0 - Personal Homepage

A demonstration of HTML & CSS.

This page serves to compelte Project0 for Harvards CS50Web MOOC. Within, I've created a few pages to showcase some of my favorite things, and push my experience with using straight HTML and CSS. Within these pages, all assignment requirements are met, including;

  • 4 complete pages with navigation between them all, utilizing Bootstrap's responsive Navbar component.
  • A few lists (like this one), tables and images.
  • A custom stylesheet, compiled from SCSS with responsive @media queries, SCSS variables, inheritence and nesting.
  • Bootstrap 4, with many uses of the package.
  • Github respository, shared through Github pages.
